# **App Name**: CommerceFlow

## Core Features:

- Product Browsing: Browse the store's product catalog with filtering and search capabilities.
- Ordering: Add items to the cart and proceed to checkout.  Minimal checkout collects item selections, names, address, and other relevant information.
- Admin Panel: A secured admin page with basic CRUD operations.
- AI-Powered Product Descriptions: Generative AI tool will assist in generating product descriptions based on product details.

## Style Guidelines:

- Primary color: Deep violet (#9466FF) for a sense of creativity, exclusivity, and forward-thinking e-commerce.
- Background color: Very light violet (#F5F0FF).
- Accent color: Soft blue (#66B2FF), analogous to violet, adding a sense of trust.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look.
- Minimal, geometric icons for categories and user actions.
- Clean, grid-based layout to showcase products effectively.
- Subtle transitions and animations on product hover and cart updates.
